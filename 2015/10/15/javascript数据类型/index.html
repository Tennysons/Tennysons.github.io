javascript数据类型

ECMAScript中严格区分大小写

使用var操作符定义的变量将成为定义该变量的作用域中的局部变量
即如果在函数中使用var定义了一个变量，那么这个变量在函数退出后就会销毁

javascript数据类型：基本类型 、 引用（对象）类型(复杂数据类型)、

基本类型与引用类型

基本类型： Undefined、String、Boolean、Null、Number
不可以给基本类型的变量添加属性和方法
引用类型： –> 对象 引用类型的值是保存在内存中的对象
可以给引用类型的变量添加属性和方法
ECMAScript不支持任何创建自定义类型的机制

typeof操作符
检测给定变量的数据类型
对一个值使用typeof操作符可能的返回值：
undefined — 该值未定义、
boolean — 该值为布尔型、
string — 该值为字符串、
number — 该值为数值型、
object — 该值为对象或null、
function — 该值为一个函数、

1、undefined类型

对于未初始化或为定义变量执行typeof操作符都会返回undefined值，但无论在什么情况下都没有必要把一个变量的值显示的设置为undefined

注意： 显示初始化变量依然是明智做法，这么做，在使用typeof操作符检测变量返回“undefined”值时就可以以此判断出被检测的变量还没有被声明

2、null类型

使用typeof操作符检测null时会返回“object”值，可将null看做空对象的指针
注意：如果定义的变量准备用来保存对象，那么最好将该变量初始化为null而不是其他值(只要意在保存对象的变量还没有真正保存对象，就应该明确的将改变量保存为null值)，这么做，只要直接检查null值就可以知道相应的变量是否已经保存了一个对象的引用

// 实际上，undefined值是派生自null值的，对他们的相等性测试返回true
alert(null == undefined); // true

3、boolean类型

该类型只有两个值：true和false （严格区分大小写，即False/True … 等都不是布尔值，只是标识符）

ECMAScript规定任何类型变量都有与这两个布尔值等价的值，可以使用Boolean()函数 将任意变量转换为相对应的布尔值

// message = "hello world";
// messageBoolean = Boolean(mwssage);
alert(messageBoolean); // true

各种数据类型及其对应的转换规则：
































数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 “”空字符串
Number 任何非零数字值（包括无穷大） 0和NaN
Object 任何对象 null
Undefined n/a undefined

4、number类型

格式：十进制、八进制、十六进制
八进制字面值的第一位必须是零（0），之后才是八进制数字序列（0~7）
注意：如果八进制字面值中的数值超出了范围，那么其前导零（0）将会被忽略，之后的数值将被当作十进制数值解析
十六进制字面值前两位必须是0x,之后跟任何十六进制数字（0~9及A~F）其中的A~F可以大写也可以小写

4.1、浮点数值（数值中必须包含一个小数点，且小数点后面必须至少一个数字）
4.2、数值范围

超出javascript数值范围的值将被自动转换成特殊的Infinity值（若是正数，为Infinity–正无穷；若为负值，则被转为-Infinity–负无穷）
若想要确定一个数值是否是有穷的，可以使用isFinity()函数 该函数会在参数位于最小与最大值之间时返回true、

4.3、NaN（Not a Number）

用于表示一个本来要返回数值的操作数未返回数值的情况（这样就避免了抛出错误）
1)、ECMAScript中任何数值除以0都会返回NaN（实际上只有0除以0才会返回NaN，正数除以0返回Infinity,负数除以0返回-Infinity）
2)、NaN与任何值都不想等，包括NaN自身

isNaN()函数，接受一个参数，该参数可以是任何类型，函数用于判断该参数是否“不是数值”
isNaN()收到一个值后，会尝试将该值转换为数值，某些不是数值的值会直接转换为数值，而任何不能被转换为数值的值都会导致该函数返回true

注意：该函数同样适用于对象

4.4、数值转换

Number()函数、适用于将任何数据类型值转换为数值类型

parseInt()函数、parseFloat()函数、专门用于将字符串转换为数值

转换规则略；

5、string类型

javascript中字符串可以用双引号括起也可以用单引号括起，二者没有区别

任何字符串的长度都可以通过访问其length属性获得

js中字符串一旦创建，其值就不可改变

把一个值转换为字符串：

方法一：toString()方法

数值、布尔值、对象、字符串值都由toString()方法
null、undefined没有toString()方法

注意：调用数值的toString()方法时，可以传递一个参数：2、8、10、16
该方法会根据传入的参数将数值转换为相应进制的字符串值

默认的（没有传参数的）输出值与指定基数10时的输出值相同
方法二：String()方法

在不知道要转换的值是不是null或undefined时，可以使用该方法
该方法可以将任何类型的值转换为字符串
转换规则：
1）、若值有toString()方法，调用该方法并返回相应的值
2）、若值为null,则返回”null”
3）、若值为undefined,则返回”undefined”

方法三：可以将要转换为字符串的值使用加号操作符与””加在一起
