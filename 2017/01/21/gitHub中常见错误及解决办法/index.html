gitHub 中常见错误及解决办法

发表于 2017-01-21 | 分类于 git

在使用gitHub过程中总会遇到一些常见的错误，以后会将经常遇到的记录到这里，以方便以后查找。

-1-


当输入：git remote add origin https://github.com/XXXX/XXXX.git
遇到错误提示：fatal: remote origin already exists.

解决办法：

  1. 先输入：$ git remote rm origin
  2. 再输入：$ git remote add origin https://github.com/XXXX/XXXX.git 即可
