javascript引用数据类型01

发表于 2017-01-08 | 分类于 javascript学习笔记

1、Object （存储和传输数据）

大多数引用数据类型都是Object类型的实例，对于在应用程序中存储和传输数据而言，是非常理想的选择

也因为大多数引用数据类型都是Object类型的实例，故对于对象检测会有以上代码中的现象

对象字面量的创建方法可以向函数传递大量可选参数

注：除非必须使用变量来访问属性，否则推荐使用点表示法

2、Array

ECMAScript数组的大小是可以动态调整的，即可以随着数据的增减自动调整长度
并且，数组中的每一项可以保存不同类型的数据

1)、length属性：保存数组的长度

特点：length属性不是只读的，即可以通过设置该属性来移除或向数组中添加新项

2)、检测数组

方法一：instanceof操作符

注意：instanceof操作符假定只有一个全局执行环境，若网页中包含多个框架，那么就会出现问题，故ECAMScript新增了isArray()方法以解决此问题

方法二：Array.isArray() 用于确定某个值到底是不是数组

3)、转换方法（toString()、valueOf()）

注：所有对象都包toLocaleString()、toString()、valueOf()方法

toString(): 调用数组的该方法会返回数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串

valueOf(): 调用数组的该方法返回的仍是数组,为创建这个字符串它会调用每一项的toString()方法

toLocaleString()：经常会返回与toString()和valueOf()相同的值，但并不总是这样
在调用数组的toLocaleString()方法时，会创建一个数组值得以逗号分隔的字符串，而与前两个方法唯一不同之处在于，这次为了取得每一项的值，调用的是每一项的toLocaleString()方法，而不是toString()方法**

调用以上三种方法，默认情况下都会以逗号分隔字符串的形式返回数组项，若使用join()方法，则可以使用不同分隔符构建这个字符串

注意：若数组的某项的值为null或undefinied,那么该值的join()、toLocaleString()、toString()、valueOf()方法返回的结果中以空字符串表示

4)、栈方法与队列方法

栈方法(LIFO)：push()方法与pop()方法

push()方法可以任意接受任意数量的参数，并把它们逐个添加到数组末尾，并返回修改后的数组长度
pop()方法则从数组的末尾移除最后一项，减少数组的length值，并返回移除的值

队列方法(FIFO):push()方法、shift()方法与unshift()方法

shift()方法能够移除数组中的第一项并返回该项的值，同时将数组长度length减1
unshift()方法与shift()方法相反，能够在数组前端添加任意个项并返回数组的长度，因此使用unshift()方法与pop()方法可以反向模拟队列，即在数组的前端添加项，末尾移除项
