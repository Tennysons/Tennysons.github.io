面向对象的特征

发表于 2017-01-07 | 分类于 java学习笔记

面向对象编程有三个特征： 封装、继承、多态
1、封装：类的抽象是指将类的实现和类的使用分离，实现的细节被封装并且对用户隐藏，这被称为类的封装，故类也被称为抽象数据类型（Abstract Data Type），通过类的封装，将类的开发与使用相互分离开
2、继承：使用已存在的类（父类）的定义作为基础建立新类（子类）的技术，新类的定义可以增加新的数据或新的功能（重写），也可以用父类的功能，但不能选择性地继承父类。通过使用继承我们能够非常方便地复用代码，能够大大的提高开发的效率
3、多态：指对象能够有多种形态,在OOP中最常用的多态性发生在当父类引用指向孩子类对象时,可以理解为一个事物的多种形态

梁漱溟自述：我的努力与反省

发表于 2016-03-22 | 分类于 若有所思

学问毕经自己求得来者，方才切实有用
学校教育不过给学生开一个端，使他更容易自学而已
自学的根本：一片向上心
一方面是父亲和彭公他们的人格感召，使幼稚的心灵隐然萌露对社会对国家的责任感，而鄙视那般世俗谋衣食求利禄的“自了汉”生活。更一方面是从那维新前进的空气中，自具一种迈越世俗的见识主张，使我意识到世俗之人虽不必是坏人，但缺乏眼光见识，那就是不行的；因此，一个人必须力争上游。顷所谓一片向上心，大抵在当时便是如此
所谓自学，应当是一个人整个生命上的向上自强，要紧在生活中有自觉。单是求知识，却不足以尽自学之事。在整个生命的向上自强中，可以包括求知识。
自学最要紧的在生活中有自觉。读书不是第一件事，第一件事，却是照顾自己的生体而如何善用它。——用它来做种种事，而读书是其中一种。
由于向上心，我常有自课与自己的责任，不论什么事很少要人监督。……真的自学，必从这里说起。
自学就是一个人整个生命的向上自强，要紧在生活中的自觉。
